TECHNICAL FEASIBILITY — IMPORTANT NOTE
----------------------------------------------------------------
The customer's original Excel file listed several outcomes that are NOT
M365 Copilot capabilities. Each use case page now includes an explicit
"Feasibility Check" panel separating:
  ✅ What Copilot can do today
  🚧 What needs other tools (Power Platform, Dynamics 365, Viva,
     Azure ML, Adrenalin HRMS API, etc.)

Examples of items moved to "Beyond Copilot":
- AI-conducted structured digital interviews with competency scoring
  → not a Copilot capability; needs a vendor or Copilot Studio agent
- Live Adrenalin HRMS connectivity
  → needs Power Automate / a custom connector
- Shift roster optimisation with hard constraints
  → Copilot is not an optimiser; needs Microsoft Shifts, D365 Field
    Service, or a WFM tool
- Predictive attrition / ROI modelling
  → needs Power BI + Azure ML or Microsoft Fabric
